Panel controls and displays reference

This section is a reference of each working switch and each display on the Project Apollo panels. It's intended mainly as a developer reference but of course it's also useful for users just in case they are curious about all these switches.

Controls and displays are presented in form of a tabulated list in numerical order by panel and area number. These tables are strongly related to the Apollo Operations Handbook, see below for more details. Each spacecraft has it's own reference table. The following is a detailed explanation of the columnar data presented in the tabulated lists:

Name

Given the exact nomenclature of a particular control or display. The name of the switch or control in the C++ code should be similar like shown in the following example: The name of the CABIN FAN switches in the code are CabinFan1Switch and CabinFan2Switch.
Panel Locator The panel locator provides the "coordinates" of the display or control. Please click on the image to the right to download the related figures from the Apollo Operations Handbook. Firstly there is the panel number, larger panels are further divided into areas. In most instances, the area correlates to a specific spacecraft system. Panels 1 through 16 have grid reference numerical marks horizontally and alphabetic marks vertically on the drawing.
Function Describes the function of the switch or display.  
Remarks Any useful comment. Please notice especially differences to the original function, bugs or other issues.
